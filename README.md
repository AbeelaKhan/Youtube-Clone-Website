# Youtube-Clone-Website
A responsive YouTube clone built with pure HTML and CSS — no JavaScript, no frameworks, no dependencies.
The project replicates YouTube's front-end UI, featuring a clean video grid layout where each video thumbnail links directly to its corresponding YouTube video.
🛠️ Built With

-HTML
-CSS

✨ Features

-YouTube-inspired UI with video thumbnail grid
-Clickable video cards that redirect to the actual YouTube video
-Fully responsive layout

🎯 Purpose
This project was built to strengthen front-end development skills by recreating a real-world interface using only core web technologies.
